# Obscura 完全ユーザーガイド

Obscura は、PC内に散らばった動画や音声ファイルを直感的に管理し、素早く検索・再生するための Eagle ライクなメディアプレイヤーです。
このガイドでは、Obscuraの **すべての機能と画面構成** について詳しく解説します。

## 1. 画面構成と基本UI

Obscuraのメイン画面は大きく3つのペイン（領域）に分かれています。

- **左サイドバー（ナビゲーション）**:
    - スマートフォルダー（未分類、タグなし、最近使用など）
    - 作成したフォルダー階層
    - すべてのタグ一覧
    - 追加したリモートライブラリの一覧が表示されます。
- **中央メインペイン（メディア一覧）**:
    - サムネイルが並ぶグリッド領域です。
    - 右上のアイコンから **グリッド表示** と **リスト表示** を切り替えることができます。
- **右インスペクタ（詳細情報）**:
    - 選択したメディアのプロパティ（タイトル、評価、メモ、タグ、拡張子情報など）を表示・編集できます。
    - ライブラリ情報、リモートライブラリの接続ユーザーを確認できます。

## 2. ライブラリの管理（作成・共有）

すべてのデータは「ライブラリ（`.library` フォルダー）」内にデータベース（SQLite3）およびコピーされたメディアファイル群として保存されます。

### ライブラリの切り替えと作成

- **新規ライブラリ**: サイドバー左上の「現在のライブラリ名」をクリックし、「新しいライブラリを作成...」から保存場所と名前を指定します。
- **既存ライブラリ**: 以前作成した `.library` フォルダーを指定して開きます。履歴（10件まで保持）からのワンクリック切り替えも可能です。

## 3. メディアの追加と削除

### ドロップ＆インポート

- エクスプローラーから動画（`.mp4`, `.mkv`, `.avi` 等）や音声（`.mp3`, `.wav`, `.flac` 等）をウィンドウ中央に **ドラッグ＆ドロップ** するとインポートが始まります。
- インポート時に動画の長さ（Duration）や解像度、サムネイルが自動的に生成されます。
- **ドットファイル（隠しファイル風の命名）** も対応しています（ただし `.git` などシステムファイルは無視されます）。

### ゴミ箱と完全削除

- メディアを削除（右クリックメニュー等）すると、まずは「ゴミ箱（Trash）」に移動します。
- **ゴミ箱から空にする**: サイドバーの「ゴミ箱」を開き、手動でファイルを「復元」するか、「完全に削除」してディスク容量を空けることができます。

## 4. メディアの分類・整理

### フォルダー (Folders)

PC内の階層に似たフォルダーツリーを作成し、ファイルを整理できます。

- サイドバーの「＋」アイコンで新規作成し、右クリックから「サブフォルダー」を作って階層化できます。
- メディアをフォルダーへドラッグすることで、所属タグを切り替えられます。

### インスペクタ編集 (Metadata Inspector)

画面右側パネルで選択したファイルの情報を詳細に変更できます。

- **タイトルと説明 (Memo)**: ファイル名をそのまま表示するか、好みのタイトルに変更可能。詳細な説明文も記入できます。
- **評価 (Rating)**: 1〜5個の「星」をつけて管理。
- **アーティスト (Artist)**: 楽曲や動画の製作者・アーティスト名を登録。yt-dlpで--embed-metadataを有効にしていると自動で入力されます。
- **関連作品 (Relations) ピッカー**: 他のメディアファイルと「親・子関係」や「シリーズ」として紐付けることができます。
- **URL**: メディアの取得元URLなどを保存。yt-dlpで--embed-metadataを有効にしていると自動で入力されます。

### タグ (Tags) 機能

- メディアに対し複数のタグを付与できます（例: `旅行`, `2024年`）。

## 5. 検索・フィルター・並び替え

### 豊富な検索機能

画面上部のヘッダーバーから、ライブラリ全体を跨いだ絞り込みが可能です。

- **テキスト検索**: ファイル名や説明（Memo）に含まれる文字を検索。
- **種類による絞り込み**: 動画 (Video) または 音声 (Audio) のみに限定。
- **タグ拡張フィルター**: 複数のタグを「AND（すべて含む）」「OR（いずれかを含む）」で高度に絞り込み。
- **レーティング/アーティスト検索**: 指定した星の数やアーティスト名での抽出フィルター。

### スマートブラウジング機能

サイドバー内に用意された自動フォルダーです。分類漏れを防ぐのに役立ちます。

- **未分類**: どのフォルダーにも属していないメディア
- **タグなし**: タグが1つも付与されていないメディア
- **最近使用**: 直近でインポートしたり、再生したメディア
- **ランダム（運任せ）**: ライブラリ全体からランダムなメディアを表示。何を観るか迷った時に。

### 表示順の保存

各フォルダーごとに、リストの並び順（追加日、再生日時、ファイル名、サイズ順など）および「昇順・降順」を個別に記憶します。

## 6. プレイヤーと再生機能

メディアをダブルクリックすると「プレイヤーモード」が起動します。

- **対応フォーマット**: 主要フォーマット（MP4, MKV, AVI, MOV, FLV, WMV, WebM, MP3, FLAC, M4A, WAV等）に対応。
- **ループ再生**: 1ファイルのリピート、またはリスト全体のリピートが可能。
- **自動再生**: 再生終了後次の動画を自動で再生開始します。
- **ピクチャー・イン・ピクチャー (PiP)**: 画面の隅に常に最前面のミニプレイヤーを表示しながら他の作業ができます。
- **ハードウェアアクセラレーション / アップスケール（高画質化）**:
    - オーディオ/ビデオ設定タブから、GPUを活用したサムネイルレンダリングや、設定されたアップスケーリングロジックを許可できます。

## 7. 高度な機能（ネットワーク共有・リモートライブラリ）

大容量のファイルを別PCに置いたまま、ネットワーク経由で安全にストリーミングできます。

### ホスト（公開側）の設定

1. `Ctrl+,` で環境設定を開き、「ネットワーク共有」＞「ホスト設定」で **共有を有効** にします。
2. 内部ではパスワード付きのエンドポイント (`/api/health`, `/api/media`, 等) が立ち上がります。
3. **共有ユーザーの管理**: 追加したい相手（クライアント）の「ユーザートークン」を登録し、「アクセストークン」を発行します（読み取り専用・フルアクセス等の権限も設定可）。

### クライアント（閲覧側）の設定

1. 同じく「ネットワーク共有」＞「クライアント設定」を開き、自分の PC の「ユーザートークン」をコピーし、ホストへ送ります。
2. ホストから渡された「アクセストークン」と、「LAN内のURL（例: `http://192.168.1.10:3000`）」を入力し **接続テスト** を実施。
3. 「追加」を押すと、左サイドバー下部にリモートライブラリが現れ、ローカルと同様に動画をクリックしてストリーミング再生が可能です。（認証エラー時には「アクセストークンが異なります。」と即座にフィードバックされます）

## 8. 環境設定とカスタマイズ

メニューアイコンや `Ctrl+,` で開く **環境設定ダイアログ** からアプリ全体をカスタマイズできます。

- **重複ファイルのチェック**:
  ヘッダーの「…」アイコン（サブメニュー）から「重複を検索」を選ぶと、ライブラリ内の完全一致ファイルを洗い出し、一括で解決（不要ファイルの削除）を行う専用画面が開きます。
- **インターフェース / テーマ設定**:
    - アプリ全体にダークテーマを採用していますが、設定画面からCSS変数を直接編集する「テーマテンプレート」機能が備わっています。
    - ハイライト色（プライマリカラー）や、背景色をお好みの色味に調整できます。
- **オーディオ出力ルーティング**:
    - 再生音を出力するスピーカー（ヘッドホンやモニター等）をアプリケーション内で独立して指定する専用ダイアログを備えています。
- **起動時の振る舞いとショートカット**:
    - 起動時に前回開いていたライブラリを自動で開く機能や、コンテキストメニューの適切な配置（画面からのはみ出し防止）など、細かなUX調整が含まれています。
    - スペースキーによる「再生/一時停止」、矢印キーによる「シーク」にフル対応。

## 9. プラグインとAPI（上級者向け）

Obscuraは独自のプラグインアーキテクチャを持ち、JavaScript (TS) を用いて機能を拡張できます。

- **プラグインインジェクション**: `plugins` フォルダに配置されたスクリプトを読み込み、再生プレイヤー本体のライフサイクル（メタデータのフェッチなど）に介入させることができます。
- **開発者用 API ドキュメント**:
    - 環境設定内の「開発者（Developer）」メニューから、提供されている「Plugin JS API」のドキュメントをアコーディオンメニュー形式で確認できます。APIを利用して各種ダウンローダー連携等が行えます。
