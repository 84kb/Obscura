# 貢献ガイドライン

Obscuraプロジェクトへの貢献に関心をお寄せいただきありがとうございます！
現在は開発中の**アルファバージョン**であるため、未知の不具合・パフォーマンス問題が多く存在する可能性があります。
些細なことでもIssueでのバグ報告や機能提案、プルリクエストなど、あらゆる形式の貢献を歓迎します。

## バグ報告

本アプリはアルファ版のため、想定外の動作や不具合が多発する可能性があります。不具合を発見された場合は、気軽にGitHub Issuesでご報告いただけると大変助かります。以下の情報を含めていただけるとスムーズです：

1. **バグの概要**: 何が起きたかを簡潔に。
2. **再現手順**: バグを再現するための具体的なステップ。
3. **期待される動作**: 本来どうあるべきだったか。
4. **実際の動作**: 実際に何が起きたか。
5. **環境情報**: OS、Node.jsのバージョンなど。
6. **スクリーンショット**:（もしあれば）

## 機能リクエスト

新しい機能のアイデアがある場合は、Issueを作成し、`enhancement` ラベルを付けてください。
どのような機能で、なぜそれが必要なのか、どのように動作すべきかを詳しく説明してください。

## プルリクエストの手順

1. リポジトリをフォークします。
2. 新しい機能用のブランチを作成します (`git checkout -b feature/amazing-feature`)。
3. 変更をコミットします (`git commit -m 'Add some amazing feature'`)。
4. ブランチをプッシュします (`git push origin feature/amazing-feature`)。
5. Github上で Pull Request を作成します。

**注意点**:
- 既存のコードスタイル（TypeScript, ESLint等）に従ってください。
- PRの概要には、変更内容とその理由を明確に記述してください。

## 開発環境のセットアップ

README.mdの「インストール」セクションを参照してください。

```bash
git clone https://github.com/84kb/Obscura.git
cd Obscura
npm install
npm run electron:dev
```
